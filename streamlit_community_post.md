# 📊 Sales Analytics Dashboard - Built with Streamlit & Plotly

Hi Streamlit Community! 👋

I'm excited to share my comprehensive **Sales Analytics Dashboard** that I just built and deployed using Streamlit and Plotly. This project showcases advanced data analytics capabilities with beautiful, interactive visualizations.

## 🚀 **Live Demo**
**GitHub Repository**: https://github.com/ERMIYASZEWDU/sales-analytics-dashboard
**Live Dashboard**: https://sales-analyticsdashboard.streamlit.app/

## ✨ **What It Does**

This dashboard provides comprehensive sales analytics across 5 main areas:

### 📈 **1. KPI Overview**
- Real-time revenue metrics and growth rates
- Period-over-period comparisons
- Business performance indicators
- Channel and category performance

### 📊 **2. Sales Trends Analysis**
- Time-series visualization (daily, weekly, monthly, quarterly)
- Seasonal pattern recognition
- Growth rate analysis with moving averages
- Trend decomposition

### 🌍 **3. Regional Performance**
- Geographic revenue distribution
- City-level performance metrics
- Regional comparison heatmaps
- Market share analysis

### 🛍️ **4. Product Analytics**
- Category performance tracking
- Product lifecycle analysis
- Price vs volume optimization
- Discount impact analysis

### 👥 **5. Customer Segmentation**
- **RFM Analysis** (Recency, Frequency, Monetary)
- Automated customer segmentation (Champions, At Risk, etc.)
- **Cohort Analysis** with retention heatmaps
- Customer lifetime value insights

## 🚀 **Easy Setup**

The project includes everything needed for quick deployment:

```bash
# One-command setup
python run_dashboard.py

# Or manual setup
pip install -r requirements.txt
python data_generator.py
streamlit run app.py
```
